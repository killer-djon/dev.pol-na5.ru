Список работ для импорта можно ввести вручную или скопировать из буфера обмена в специальную текстовую область на странице «Импорт работ». Вы также можете импортировать работы из предварительно подготовленного текстового файла.

    Данные для импорта должны быть подготовлены таким образом, чтобы каждая работа располагалась на отдельной строке. При этом каждая строка должна быть одного и того же формата и может содержать несколько колонок, разделенных знаком табуляции (TAB), точкой с запятой (;) или запятой (,).

Допустимые колонки для импорта:
Описание
Дата начала
Выполнить до
Дата завершения
Подлежит оплате
Стоимость
Валюта
Назначения

    Текст в каждой колонке можно взять в кавычки. Если вы хотите использовать кавычки внутри колонки с описанием работы, то вы должны взять все описание в кавычки и перед каждой внутренней кавычкой поставить дополнительную кавычку.
    Пример: "описание с ""внутренними"" кавычками"

    Чтобы импортировать назначения, поместите идентификаторы назначенных пользователей в колонку Назначения и разделите их запятыми.
    Пример: "JOHN, bob, Susan"
